# Akande Gbolahan Emmanuel | Portfolio

Personal portfolio website showcasing blockchain development work, technical writing, and projects across 7+ blockchain ecosystems.

## Tech Stack

- **Frontend**: React 18, Tailwind CSS, Framer Motion
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom Web3 theme (dark/light mode)
- **Contact**: EmailJS integration
- **Icons**: React Icons
- **Carousel**: Swiper.js

## Features

- Dark/light mode toggle with localStorage persistence
- Responsive floating bottom navigation with scroll spy
- Animated stat counters and scroll-triggered animations
- 13 project cards with status badges and tech stack pills
- 19 published articles linked from Hashnode and Medium
- 7 blockchain ecosystem showcase
- Work experience timeline
- Contact form with EmailJS

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
  components/
    layout/      Nav, Footer
    sections/    Hero, About, Skills, WorkExperience, Portfolio,
                 Ecosystem, Services, Impact, Education,
                 Testimonials, Content, Contact
    ui/          Reusable components (Badge, Card, ProjectCard, etc.)
  data/          All content data (projects, skills, experience, etc.)
  context/       ThemeContext (dark/light mode)
  hooks/         useScrollSpy, useInView
  assets/        Images
```
